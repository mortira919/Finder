✦ Как получить ключ?
  1. Открой console.cloud.google.com — войди под своим Google аккаунтом
  2. Создай новый проект (или выбери существующий) в верхнем меню
  3. Перейди в APIs & Services → Library
  4. Найди и включи Places API (нажми Enable)
  5. Перейди в APIs & Services → Credentials
  6. Нажми Create Credentials → API Key — скопируй ключ
  7. Добавь биллинг: Billing → Link billing account (карта нужна, первые $200 бесплатно — ~5000 поисков)
⚠ Без биллинга API вернёт ошибку OR_BACR2_44
